# Spotify-Song-Popularity-Analysis-Prediction
This project analyzes Spotify song data to understand the factors that influence song popularity. Using audio features provided by Spotify, the study explores patterns in popular music and builds predictive models to estimate popularity scores. 

The goal is to combine **data analytics and machine learning** to uncover insights relevant to music analytics, recommendation systems, and digital marketing strategies.

---

## Dataset Description
The dataset includes song-level audio features such as:
- Danceability
- Energy
- Valence
- Tempo
- Loudness
- Acousticness
- Popularity score

---

## Methodology
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Feature correlation analysis
- Predictive modeling using machine learning algorithms
- Model evaluation and interpretation

---

## Tools & Technologies
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn
- Jupyter Notebook

---

## Key Insights
- Popular songs generally exhibit higher energy and danceability
- Feature combinations are more important than individual attributes
- Certain emotional tones (valence) influence popularity depending on genre

---

## Conclusion
This project demonstrates how music data can be leveraged to generate meaningful insights and predict trends. It highlights the role of analytics in understanding user preferences and supporting data-driven decisions in the music industry.
